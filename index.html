Welcome to Rehabilitation Game

An interactive rehab system for patients using sensors, ML, and games.

Features

  • Sensor-based control using wrist motion
  • ML-driven level recommendations(Coming soon...)
  • Real-time data: force, pressure, gyro & more
  • Custom therapy sessions with progress tracking

How It Works

Sensor & Hardware

Uses Arduino Leonardo, motor encoder & sensors (FSR, Gyro, Accel) to capture wrist motion.

ML Prediction

ML algorithm evaluates performance and predicts next rehab level

Interactive Game

The game adapts in difficulty and encourages recovery through gameplay.
